# user_top Gate-Level + SDF Simulation

This directory is reserved for post-route gate-level simulation artifacts.

- `logs/`: simulator stdout/stderr summaries
- `waves/`: optional VCD/FST/FSDB outputs
- `run_gatelevel_sdf_template.sh`: template entrypoint for SDF-annotated simulation using `results/user_top_postroute.v/.sdf`
