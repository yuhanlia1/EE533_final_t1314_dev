# Legacy Software References

This directory keeps historical `/sw` resources that are no longer part of the
formal v7 software flow.

Current supported user-facing tools live at:

- `sw/annctl`
- `sw/cpuctl`
- `sw/gpuctl`
- `sw/annmodelctl`
- `sw/cpu_toolchain/`
- `sw/gpu_toolchain/`
- `sw/model_toolchain/`
- `sw/reg_defines_v7.h`

The subdirectories moved here are reference-only. They are not part of the
default regression or the primary CPU/GPU programming workflow.
